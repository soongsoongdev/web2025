Chapter 4 - 함수와 스코프

1. 함수란?

함수는 특정 작업을 수행하는 코드 블록으로, 반복되는 동작을 효율적으로 처리하거나 코드를 구조화할 때 사용됩니다. 함수로 나누면 코드가 더 보기 좋고, 유지보수가 쉬워집니다.

함수 정의와 실행

function printMessage() {
  console.log("이건 함수로 출력한 메시지입니다.");
}
printMessage();  // 함수 호출

매개변수와 반환값

함수는 외부에서 값을 받아 내부 로직에 사용하고, 결과를 반환할 수도 있습니다. 

function getSquare(number) {
  return number * number;
}
console.log(getSquare(5)); // 25 출력

기본값 설정

매개변수에 값이 전달되지 않으면, 미리 지정한 기본값을 사용할 수 있습니다. 

function greet(name = "손님") {
  console.log(`${name}님, 환영합니다!`);
}
greet();           // 손님님, 환영합니다!
greet("지민");      // 지민님, 환영합니다!

2. 변수의 유효범위 (Scope)

변수는 선언된 위치에 따라 접근 가능한 범위가 달라집니다.

지역 스코프

함수 안에서 선언된 변수는 외부에서 접근할 수 없습니다. 

function sayHi() {
  let message = "안녕하세요";
  console.log(message);
}
// console.log(message); // 오류 발생

블록 스코프

let과 const는 블록({}) 내부에서만 유효하지만, var는 블록을 무시하고 함수 전체에서 사용될 수 있습니다. 

if (true) {
  let food = "김밥";
  var drink = "콜라";
}
// console.log(food); // 에러
console.log(drink);   // 콜라

전역 스코프

스크립트의 바깥에서 선언한 변수는 어디서든 사용할 수 있지만, 무분별한 사용은 피하는 게 좋습니다. 

let theme = "dark";

function changeTheme() {
  theme = "light";
}
console.log(theme); // dark
changeTheme();
console.log(theme); // light

3. 익명 함수와 표현식

이름이 없는 함수를 변수에 저장해서 사용하는 방식입니다. 

const add = function(x, y) {
  return x + y;
};
console.log(add(3, 7)); // 10

4. 즉시 실행 함수 (IIFE)

함수를 선언과 동시에 실행하고 싶을 때 사용하는 패턴입니다. 

(function(name) {
  console.log(`${name}님 반가워요.`);
})("혜린");

5. 화살표 함수

ES6부터 도입된 짧은 문법의 함수 선언 방식입니다. 

const multiply = (a, b) => a * b;
console.log(multiply(3, 4)); // 12

const sayHello = () => "안녕하세요!";
console.log(sayHello());

6. 콜백 함수

콜백 함수는 다른 함수에 인수로 전달되어 특정 시점에 실행되는 함수입니다.

7. 전개 구문 (...)

배열이나 객체의 요소를 하나씩 펼쳐서 사용할 수 있는 문법입니다. 

const pets = ["강아지", "고양이", "햄스터"];
console.log(...pets); // 강아지 고양이 햄스터

function sumAll(...numbers) {
  return numbers.reduce((acc, cur) => acc + cur, 0);
}

console.log(sumAll(1, 2, 3, 4)); // 10

8. 타이머 함수

일정 시간 간격으로 실행하거나, 특정 시간 후에 한 번 실행할 수 있습니다.

setInterval / clearInterval

let count = 0;
const intervalId = setInterval(() => {
  console.log("반복 실행 중...");
  count++;
  if (count === 3) {
    clearInterval(intervalId);
    console.log("반복 종료");
  }
}, 1500);

setTimeout

setTimeout(() => {
  console.log("3초 후 실행됩니다.");
}, 3000);
