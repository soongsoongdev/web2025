5주차 - DOM과 웹 요소 제어

1. DOM(Document Object Model)과 DOM 트리

DOM은 HTML 문서를 **객체로 표현한 구조**로, 자바스크립트를 통해 문서의 각 요소에 접근하거나 조작할 수 있게 해줍니다.
이 구조는 트리 형태로 이루어져 있어 루트 노드(document)를 시작으로, 부모/자식/형제 노드 간의 관계를 가집니다.

2. 웹 요소 선택하기

DOM 요소에 접근하려면 CSS 선택자와 함께 아래 메서드를 사용합니다.

선택자 의미 예시
태그 선택자 해당 태그의 모든 요소 p
클래스 선택자 특정 클래스에 속한 요소 .highlight
ID 선택자 해당 ID를 가진 단일 요소 #main
하위 선택자 상위 요소 내부의 모든 하위 요소 div span
자식 선택자 바로 아래 자식 요소만 선택 ul > li

요소 접근 메서드

  • document.querySelector("선택자") – 첫 번째 일치하는 요소
  • document.querySelectorAll("선택자") – 모든 일치 요소 (NodeList로 반환)
  • document.getElementById("id") – ID로 요소 선택
const title = document.querySelector("#title"); // id="title" 요소를 선택

3. 웹 요소의 내용 확인 및 수정

선택한 요소의 내용을 가져오거나 바꿀 수 있습니다.

  • innerText: 브라우저 화면에 보이는 텍스트만 가져옴
  • textContent: 숨겨진 텍스트, 공백 포함 전체 텍스트
  • innerHTML: HTML 태그를 포함한 전체 콘텐츠
  • .value: form 요소에 입력된 값을 가져옴

※ 내용 변경:

title.innerText = "새로운 제목입니다";
document.querySelector("img").src = "newImage.jpg";

4. 이벤트를 이용한 요소 조작

클릭 등의 사용자 동작에 반응하도록 이벤트 리스너를 설정할 수 있습니다.

const title = document.querySelector("#title");

title.onclick = () => {
  title.classList.add("clicked"); // clicked 클래스가 추가됨
};

클래스 조작은 자주 사용됩니다:

  • element.classList.add("클래스명")
  • element.classList.remove("클래스명")
  • element.classList.contains("클래스명")
  • element.classList.toggle("클래스명") – 있으면 제거, 없으면 추가

※ 예시: 클릭할 때마다 클래스가 토글됨

const btn = document.querySelector("button");
btn.onclick = () => {
  document.body.classList.toggle("dark");
};

5. 스타일 조작하기

자바스크립트로 직접 요소의 CSS 속성을 바꿀 수도 있습니다.

const title = document.querySelector("#title");
title.onclick = () => {
  title.style.color = "white";
  title.style.backgroundColor = "black";
};

6. 폼(Form) 요소 제어

사용자가 입력한 데이터를 가져오거나, 특정 항목을 선택했을 때 동작을 수행할 수 있습니다.

  • document.querySelector("#id").value: 입력 필드 값 가져오기
  • document.forms[0].elements[1]: 첫 번째 폼의 두 번째 필드
  • select.options / select.selectedIndex: 선택 목록의 현재 선택 항목
  • input[type='radio']:checked: 선택된 라디오 버튼 접근
