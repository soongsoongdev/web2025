Chapter 3 - 연산자와 제어문

1. 연산자

연산자는 프로그램의 계산이나 논리를 수행할 때 사용하는 기호입니다.

  • 산술 연산자: + - * / % ++ --
  • 할당 연산자: = += -= *= /=
  • 비교 연산자: == != < > === !==
  • 논리 연산자: && || !
  • 문자열 연결 연산자: +

===는 값뿐 아니라 자료형까지 일치해야 true를 반환합니다. 예: 3 === "3"은 false입니다.

let x = 10;
let y = 4;
let result = x + y--;  // result: 14, y: 3
result = ++x - y;      // x: 11, y: 3, result: 8

2. 조건문

조건문은 프로그램의 흐름을 조건에 따라 다르게 만들 수 있습니다.

  • if / else 문
  • 삼항 연산자: (조건) ? 참일 때 : 거짓일 때
  • switch 문: 여러 경우에 따라 분기

3. 반복문

동일한 작업을 여러 번 수행할 때 사용하는 구조입니다.

  • for: 기본 반복문
  • while / do...while: 조건 반복
  • forEach: 배열 요소 반복
  • for...in: 객체 속성 반복
  • for...of: 배열, 문자열 반복
for (let i = 1; i <= 3; i++) {
  console.log("반복:", i);
}

const students = ["Kim", "Lee", "Park"];
students.forEach(function(name) {
  console.log(name + " 학생입니다.");
});

4. 반복 제어문

  • break: 반복문을 중간에 종료
  • continue: 현재 반복만 건너뛰고 다음 반복으로
for (let i = 1; i <= 5; i++) {
  if (i === 3) continue; // 3은 건너뜀
  if (i === 5) break;    // 5에서 반복 종료
  console.log(i);
}
