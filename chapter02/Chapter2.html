Chapter 2 / 변수와 자료형

1. 변수란?

  • 값을 저장하기 위한 이름이 있는 공간
  • 변수 이름은 숫자로 시작 불가, 공백 금지, 대소문자 구분
  • 예약어는 변수명으로 사용 불가

변수 선언

  • let: 변수 선언, 재할당 가능
  • const: 상수 선언, 재할당 불가능
  • var: 재선언과 재할당 가능하나 호이스팅 이슈로 사용 지양
// 예시
let age = 20;
const birthYear = 2000;
var name = "철수";

2. 입출력 함수

  • alert(): 알림 창
  • confirm(): 확인 창 (true/false 반환)
  • prompt(): 입력 창 (문자열 반환)
  • console.log(): 콘솔에 로그 출력
  • document.write(): HTML 문서에 직접 출력
let name = prompt("이름을 입력하세요");
console.log(`${name}님, 환영합니다!`);

3. 자료형(Data Types)

자바스크립트는 동적 타입 언어로, 변수의 자료형이 자동으로 결정됨

  • 원시 자료형: number, string, boolean, null, undefined, symbol
  • 참조 자료형: object, array, function 등
  • typeof() 함수로 자료형 확인 가능
typeof(10);         // "number"
typeof("10");       // "string"
typeof(true);       // "boolean"

4. 템플릿 리터럴

`문자열 ${변수}` 형식으로 변수와 문자열을 쉽게 조합할 수 있음

let name = "백두산";
let room = 201;
console.log(`${name}님, ${room}호 강의실로 입장하세요.`);

5. 객체와 배열

객체

let book = {
    title: "자바스크립트 입문",
    pages: 300,
    isFinished: true
};
console.log(book.title);           // "자바스크립트 입문"
console.log(book["isFinished"]);  // true

배열

let colors = ["red", "blue", "green"];
console.log(colors[1]); // "blue"

심벌(Symbol)

고유한 식별자가 필요한 경우에 사용

let id = Symbol();
const member = {
    name: "lee",
    [id]: 01
};
console.log(member[id]);

6. 자료형 변환

  • Number(): 숫자로 변환
  • parseInt(), parseFloat(): 문자열을 정수/실수로 변환
  • toString(), String(): 문자열로 변환
  • Boolean(): 논리형으로 변환
Number("20");         // 20
parseInt("10.5");     // 10
(10).toString();      // "10"
Boolean(0);           // false
Boolean("hello");     // true
